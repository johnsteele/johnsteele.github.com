Eclipse Manager: An Application for Managing Eclipse.

May 11th, 2013

Managing Eclipse distributions, installations, and workspaces is a common problem among Eclipse users. Eclipse Manager aims to help solve this problem by providing a single point of access for these Eclipse artifacts. Eclipse Manager is a fat client, which allows users to store the Eclipse artifacts on different machines.

You can find the source code for Eclipse Manager on github.
